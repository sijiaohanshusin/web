硬件基础知识

// HARDWARE 101
常用电子元件

电阻器

限制电流的元件，单位是欧姆（Ω），用于控制电路中的电压和电流。遵循欧姆定律 V = I × R。常见类型：碳膜电阻、金属膜电阻、绕线电阻、可变电阻（电位器）。

电容器

储存电荷的元件，单位是法拉（F），特点是「阻直流、通交流」，用于滤波、耦合、定时等电路。常见类型：陶瓷电容、电解电容、薄膜电容、可变电容。

电感器

储存磁能的元件，单位是亨利（H），特点是「阻交流、通直流」，用于滤波、振荡、变压器等电路。常见类型：空芯电感、铁氧体磁芯电感、铁芯电感、可调电感。

二极管

单向导电的半导体器件：正向导通、反向截止，硅管正向压降约 0.7V。用于整流、检波、稳压等电路。常见类型：整流二极管、稳压（齐纳）二极管、发光二极管（LED）、肖特基二极管。

晶体管

放大和开关信号的半导体器件，是现代电子设备的核心元件。三种工作状态：截止、放大、饱和。常见类型：双极型晶体管（BJT）、场效应晶体管（FET）、MOSFET、IGBT。

运算放大器

调节和放大模拟信号的器件，接入适当的反馈网络后可以实现放大、滤波、比较等各种功能，特点是高输入阻抗、低输出阻抗、增益可调。协会第二次硬件培训的主角就是它。

电路图与电路板

电路图示例

印刷电路板（PCB）

把电路「画」在覆铜板上，是电子制作的最终形态。协会会教你从原理图绘制到 PCB 打样的完整流程。

面包板

虽然它叫这个名字……但它可是不能吃哦。无需焊接即可快速搭建电路原型，新手实验必备。

洞洞板

比面包板更牢固、比 PCB 更灵活的过渡方案，练习焊接的好伙伴。

腐蚀板

用化学腐蚀法自制的电路板，科协的传统手艺。

单片机（学习重点）

单片机（MCU）又称单片微控制器，它不是完成某一个逻辑功能的芯片，而是把一个微型计算机系统集成到一个芯片上：拥有丰富的 IO 端口，适合控制工作；部分高性能单片机还拥有较强的信号处理能力，可部署边缘 AI 算力。

嵌入式开发环境

KEIL

老牌单片机 C 语言开发系统，几乎适配市面所有单片机型号。

VS Code

轻量级代码编辑器，插件生态丰富，与 KEIL 搭配能提供更现代的编码体验。

CLion

跨平台 C/C++ 集成开发环境，配合 STM32CubeMX 和 CMake，适合 STM32 系列开发。

CCS

TI 公司推出的集成开发环境，自带图形化配置页面，主要用于 TI 单片机（如 MSPM0 系列）。

STM32CubeIDE

ST 公司推出的集成开发环境，自带图形化配置页面，主要用于 STM32 系列单片机开发。

STM32CubeMX

ST 公司的图形化配置工具，可与 KEIL、VS Code、CLion 搭配使用。

串口助手

串口调试工具：单片机 UART 通信经过 USB 转 TTL 后即可与电脑建立通信。

软件常用工具

烧录器（仿真器）

用来把程序烧录进单片机，并对程序进行在线调试。

逻辑分析仪

抓取信号并解码通信协议，检查信号与协议问题的利器。

USB 转 TTL

连接单片机与电脑，把单片机 UART 通信的内容传给电脑上的串口助手。
