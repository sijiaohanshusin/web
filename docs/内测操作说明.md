【HEU ESTA 官网 & 论坛内测开始啦】

各位同学，科协新版官网和会员论坛开始内部测试，欢迎大家试用找茬！

一、两个地址

官网：https://heuesta.cn
论坛：https://bbs.heuesta.cn

二、怎么开始（3 步）

1. 打开官网，点右上角「注册」，填写学号、姓名、学院、年级（QQ 选填），设置用户名和密码
2. 注册后在群里说一声「已注册，用户名是 xxx」，管理员通过审核后你会能正常登录
3. 登录官网后，直接点导航栏「论坛」——会自动登录，不需要再注册一遍

三、可以玩什么

官网：
- 首页：招新视频、图片走廊、B 站最新培训视频（手机也试试）
- 新生指南：完整的新生扫盲内容（元件/工具/软件/培训安排）
- 学习中心：21 章电子学教材 + 6 篇附录
- 资料站：登录后可见「仅会员」资料，可搜索下载
- 个人中心：改资料、传头像、改密码（都在官网操作）

论坛：
- 六个版块随便逛：新生答疑 / 硬件电路 / 嵌入式与软件 / 竞赛专区 / 畅所欲言 / 公告板
- 去「新人报到楼」盖个楼刷存在感
- 发帖支持 Markdown、表情、直接粘贴图片、@别人
- 你在协会的身份（会员/干事/管理员）会自动显示为帖子旁的彩色头衔

四、发现问题怎么反馈

论坛「畅所欲言」版块置顶了「🛠️ 内测问题反馈 & 建议收集楼」，按帖子里的格式盖楼即可。
Bug、建议、吐槽都收，截图最好。提交前先 Ctrl+F5 强刷一次排除缓存。

五、注意事项

- 内测期间数据可能调整清理，重要内容请自己留底
- 手机、平板、电脑都欢迎测试，浏览器兼容问题也算 Bug

有问题群里喊，感谢大家！
